# OVCP admin UI

Svelte 5 + Vite. See the [repo README](../../README.md) and `man ovcp` for
the full picture — this UI is one piece of a single Go binary, not a
standalone app.

```sh
npm run dev     # dev server (proxy the API or run `ovcp serve` alongside)
npm run build   # → web/dist, embedded into the ovcp binary
```
